# Ligeira Telecom

Site oficial da **Ligeira Telecom**, provedor de internet 100% fibra óptica atendendo a região do Cariri cearense (Juazeiro do Norte, Brejo Santo, Mauriti, Milagres e Penaforte).

## 🚀 Tecnologias Utilizadas

*   [Vite](https://vitejs.dev/) — Ferramenta de build rápida para o front-end
*   HTML5 / CSS3 (Vanilla CSS com variáveis de Design System)
*   JavaScript (ES6+)
*   [Lucide Icons](https://lucide.dev/) — Biblioteca de ícones moderna

## 🛠️ Como Executar Localmente

### Pré-requisitos
Certifique-se de ter o [Node.js](https://nodejs.org/) instalado na sua máquina.

### Passos para rodar
1. Instale as dependências do projeto:
   ```bash
   npm install
   ```
2. Execute o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
3. Acesse `http://localhost:5173` no seu navegador.

### Build de Produção
Para compilar e otimizar o projeto para produção:
```bash
npm run build
```
Os arquivos gerados para publicação ficarão na pasta `dist/`.
